Le formulaire suivant vous aidera à créer un modèle de fichier de configuration pour votre serveur. Les modalités de vérification sont les mêmes que par le bot, donc si vous avez des erreurs, vous pouvez les corriger ici plutôt que manuellement.
Ensuite, il vous suffira de cliquer sur "**Télécharger le modèle**" pour obtenir un fichier de configuration prêt à être utilisé.

:::note
Le bouton "**Télécharger le CSV**" permet d'obtenir un fichier permettant d'importer plusieurs personnages en même temps.
Pour plus d'information, vous pouvez vous référer à la [documentation](../sheet/import_export.md).
:::
